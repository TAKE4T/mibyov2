🔍 未病診断フロー デバッグテスト

テスト手順

  1. ローカル開発サーバーを開く
  2. ブラウザの開発者ツール（F12）を開く
  3. Consoleタブを選択
  4. 診断を開始して各ステップを確認

チェックポイント

📋 確認項目：
✅ 1. 挨拶メッセージが表示される
✅ 2. 「診断を開始する」ボタンが表示される
✅ 3. ボタンクリックで最初の質問が表示される
✅ 4. 選択肢をクリックできる
✅ 5. 次の質問に正しく進む
✅ 6. 質問が重複しない
✅ 7. 複数選択質問で「次へ」ボタンが機能する
✅ 8. 全27問が順番に表示される
✅ 9. 最後に結果画面が表示される

コンソールログの監視

🔍 注目するログ：
• [ChatDiagnosis] State transition: welcome, Question: 0/27
• [ChatDiagnosis] State transition: ready, Question: 0/27
• [ChatDiagnosis] State transition: asking, Question: 0/27
• [ChatDiagnosis] Option clicked: [選択肢], Current state: waiting_answer
• エラーメッセージがないか確認

既知の問題と対処法

❌ 質問が重複する場合：
状態機械の遷移ログを確認し、asking→waiting_answer→processing→askingの流れが正しいかチェック
❌ ボタンが反応しない場合：
現在の状態がwaitingでない可能性。ログで状態を確認
❌ 結果画面でエラーが出る場合：
answers配列とquestions配列の整合性を確認
